🧾System Logs

An audit trail of user actions across the system, for accountability and troubleshooting.

Frequently Asked Questions

Common questions about System Logs.

What kind of actions are recorded in System Logs?⌄

User actions across modules — creates, updates, and deletes — so changes can be traced back to who made them and when.

Can I filter logs by user or date?⌄

Yes, use the filter controls on the System Logs page to narrow results.
