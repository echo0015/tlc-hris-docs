Frequently Asked Questions

Common questions about Requests Hub.

Which request types can I file from my phone?⌄

Leave, Overtime, CTO, and OBT — the same four self-service types available on the web app.

Can I see requests filed by other employees?⌄

No — the mobile app only shows your own requests. Approvers use the web app to review requests from their team.
