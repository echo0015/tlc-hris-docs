Mobile App/Leave Requests

🗓️Leave Requests

File and track your own leave requests, and check your remaining leave credits.

Rules & Validation

  • Leave type, start date, end date, reason — all required.
  • The app checks the leave type's gender restriction, minimum service months, and your remaining balance (including any allowed negative limit) before letting you submit — the same checks the server enforces.
  • You can cancel a request yourself only while it's still Pending.

Frequently Asked Questions

Common questions about Leave Requests.

My leave request has been "Pending" for a while — what's happening?⌄

Every request needs sign-off from your Immediate Head first, then HR. It stays Pending until both steps are done.

Can I cancel a request I already submitted?⌄

Yes, but only while it is still Pending.

I don't see the leave type I want to file. Why?⌄

Some leave types have restrictions — gender, minimum months of service, or employment status — that make them unselectable if you're not eligible.
